System Setup:

sudo apt update -y

sudo apt install -y ca-certificates curl gnupg

Add Docker's Official Repo:

sudo install -m 0755 -d /etc/apt/keyrings

curl -fsSL https://download.docker.com/linux/ubuntu/gpg | sudo gpg --dearmor -o /etc/apt/keyrings/docker.gpg
echo \
  "deb [arch=$(dpkg --print-architecture) signed-by=/etc/apt/keyrings/docker.gpg] \
  https://download.docker.com/linux/ubuntu $(. /etc/os-release && echo $VERSION_CODENAME) stable" \
| sudo tee /etc/apt/sources.list.d/docker.list > /dev/null

sudo apt update -y

sudo apt install -y docker-ce docker-ce-cli containerd.io docker-buildx-plugin docker-compose-plugin

sudo systemctl enable --now docker

sudo usermod -aG docker "$USER"

sudo reboot now



Storage:

First we need to make sure that our nvr partition or external drive is mounted for use in the system. Run the following command to identify the partition's id. Should be something like /dev/sdX# (e.g. /dev/sda6).

sudo blkid
Copy the UUID value of that partition for later use.

e.g. UUID="d15d4aed-d813-4312-9c58-0dcd18972b02"

Create the mountpoint in the filesystem.

sudo mkdir -p /nvr

Add the partition or disk to the /etc/fstab:

sudo nano /etc/fstab
Add this to the end:

# ZoneMinder NVR
UUID=d15d4aed-d813-4312-9c58-0dcd18972b02       /nvr    ext4    defaults 0      2
Exit and Save:

Press CTRL + X Press Y Press Enter

Test the configuration:

sudo mount -a
df -h | grep nvr
You should see the mounted partition or drive.

Make sure ZoneMinder can write to it.

sudo chown -R <user>:<user> /nvr
sudo chmod -R 777 /nvr
ls -al /
You should see <user> as the owner of the folder now.


Deployment:

docker network create proxy

We need to create a folder for apache to override the default site.

sudo mkdir ~/Desktop/Zoneminder

nano ~/zoneminder.yaml

---
services:

  # Container #1 - Traefik Reverse Proxy
  traefik:
    image: traefik:v2.11
    container_name: Traefik
    command:
      - "--log.level=INFO"
      - "--api.dashboard=true"
      - "--providers.docker=true"
      - "--providers.docker.exposedbydefault=false"
      - "--entrypoints.web.address=:80"
      - "--entrypoints.websecure.address=:443"
      # Redirect all HTTP to HTTPS
      - "--entrypoints.web.http.redirections.entrypoint.to=websecure"
      - "--entrypoints.web.http.redirections.entrypoint.scheme=https"
      # Let's Encrypt (HTTP-01)
      - "--certificatesresolvers.letsencrypt.acme.httpchallenge=true"
      - "--certificatesresolvers.letsencrypt.acme.httpchallenge.entrypoint=web"
      - "--certificatesresolvers.letsencrypt.acme.email=<your email>"
      - "--certificatesresolvers.letsencrypt.acme.storage=/letsencrypt/acme.json"
    ports:
      - "80:80"
      - "443:443"
    volumes:
      - "/var/run/docker.sock:/var/run/docker.sock:ro"
      - "./letsencrypt:/letsencrypt"
    networks:
      - proxy
    restart: unless-stopped

  # Container #2 - Zoneminder
  zoneminder:
    image: dlandon/zoneminder:latest
    container_name: Zoneminder
    # Shared Memory for recording
    shm_size: "2g"
    environment:
      TZ: "America/New_York"
      PUID: "1000"
      PGID: "1000"
      INSTALL_HOOK: "0"
      INSTALL_FACE: "0"
      INSTALL_TINY_YOLOV3: "0"
      INSTALL_YOLOV3: "0"
      INSTALL_TINY_YOLOV4: "0"
      INSTALL_YOLOV4: "0"
      MULTI_PORT_START: "0"
      MULTI_PORT_END: "0"
    volumes:
      - "/home/meetspa/Desktop/Zoneminder:/config:rw"
      - "/nvr:/nvr:rw"
    networks:
      - proxy
    labels:
      # Enable Traefik for this container
      - "traefik.enable=true"

      # Router: HTTPS
      - "traefik.http.routers.zm.rule=Host(`<domain.tld>`)"
      - "traefik.http.routers.zm.entrypoints=websecure"
      - "traefik.http.routers.zm.tls=true"
      - "traefik.http.routers.zm.tls.certresolver=letsencrypt"

      # Service: tell Traefik which internal port ZoneMinder uses (80)
      - "traefik.http.services.zm.loadbalancer.server.port=80"
    restart: unless-stopped

networks:
  proxy:
    external: true
---

Navigate to the directory where the yaml is and run:

docker compose -f zoneminder.yaml up -d

This container installs on first run.  You can track it with:

docker logs -f Zoneminder

To clean up the project use:

docker compose -f zoneminder.yaml down


Exposing and Let's Encrypt SSL:

Point your DNS to the public IP.  In the router, port forward to the machine running Zoneminder on ports 80 and 443.
